Platform Support

GMAT has been rigorously tested on the Windows 7 platform and we perform nightly regression tests running almost 13,000 test cases for the system core and over 4000 test cases for the GUI interface.

For release R2015a, we have only addressed issues on Mac and Linux that also occur on the Windows 7 platform. We are currently addressing the few remaining critical issues on Mac and Linux and will release production quality console builds on those platforms soon after the Windows release of R2015a.
